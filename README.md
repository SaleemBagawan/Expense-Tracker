# Expense Tracker (CLI)

## Description
A simple command-line based expense tracker built using Python.
It allows users to add and view their daily expenses.

## Features
- Add expense
- View all expenses
- Simple CLI interface

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:
python main.py
